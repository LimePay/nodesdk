## Summary
-----
Limepay JavaScrip SDK provides functionality to interact easily with rest-api.
SDK functionality:
* Connecting with API
* CRUD operations over Shoppers
* Create/Read operations over Fiat/Relayed Payments
* Create/Read operations over Invoices
