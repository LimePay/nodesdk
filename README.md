## Summary
-----
Limepay JavaScrip SDK provides functionality to interact easily with rest-api.
SDK functionality:
* Connecting with API
* CRUD operations over Shoppers
* CRU operations over Fiat/Relayed Payments
* CRU operations over Invoices
